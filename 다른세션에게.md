  Git으로 하면                                                                                                                        
                                                                                                                                      
  세션 시작 → git pull (충돌 가능!)                                                                                                   
    → 충돌 나면 Claude Code 시작도 못 함                                                           
    → 수동으로 해결해야 함                     
  세션 끝 → git add + commit + push
    → 363MB .jsonl 파일 매번 커밋
    → repo 금방 몇 GB 됨

  Google Drive로 하면

  파일 수정 → 자동 업로드 (백그라운드)
  다른 PC → 자동 다운로드 (백그라운드)
  충돌 → 거의 안 남 (동시에 양쪽에서 같은 파일 수정할 때만)

  핵심 차이

  ┌───────────┬───────────────────────────┬───────────────────────┐
  │           │            Git            │     Google Drive      │
  ├───────────┼───────────────────────────┼───────────────────────┤
  │ 충돌 빈도 │ 매번 pull할 때 가능       │ 동시 편집할 때만      │
  ├───────────┼───────────────────────────┼───────────────────────┤
  │ 충돌 해결 │ 수동 merge                │ 최신 파일 우선 (자동) │
  ├───────────┼───────────────────────────┼───────────────────────┤
  │ 용량      │ .jsonl 히스토리 계속 쌓임 │ 최신 버전만 유지      │
  ├───────────┼───────────────────────────┼───────────────────────┤
  │ 자동화    │ 훅 짜야 함                │ 설치만 하면 끝        │
  └───────────┴───────────────────────────┴───────────────────────┘

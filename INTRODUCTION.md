# 데비 & 마를렌 봇

이터널리턴 캐릭터 '데비'와 '마를렌'을 테마로 한 Discord 봇입니다.

---

## 주요 기능

### 전적 검색
| 명령어 | 설명 |
|--------|------|
| `/이터널리턴 전적 <닉네임>` | 플레이어 전적 조회 (티어, MMR, 승률, 최근 경기) |
| `/이터널리턴 통계` | 캐릭터별 픽률, 승률, 평균 순위 통계 |

### 음성 채널 (TTS)
| 명령어 | 설명 |
|--------|------|
| `/음성 입장` | 봇이 음성 채널에 입장 |
| `/음성 퇴장` | 봇이 음성 채널에서 퇴장 |
| `/음성설정 <채널>` | TTS를 사용할 채널 지정 |

- 데비/마를렌 캐릭터 목소리로 채팅을 읽어줍니다
- AI 음성 합성(TTS) 기술 사용

### 음악 재생
| 명령어 | 설명 |
|--------|------|
| `/음악 재생 <URL 또는 검색어>` | YouTube 음악 재생 |
| `/음악 스킵` | 다음 곡으로 넘기기 |
| `/음악 정지` | 재생 중지 |
| `/음악 대기열` | 현재 재생 목록 확인 |

### YouTube 알림
| 명령어 | 설명 |
|--------|------|
| `/유튜브 알림 받기` | 새 영상 업로드 시 DM 알림 |
| `/유튜브 알림 안받기` | 알림 해제 |

- 이터널리턴 공식 YouTube 채널 새 영상 자동 알림
- 5분마다 새 영상 확인

### 기타
| 명령어 | 설명 |
|--------|------|
| `/피드백 <내용>` | 개발자에게 피드백/건의사항 전송 |
| `/설정` | [관리자] 서버 알림 채널 설정 |
